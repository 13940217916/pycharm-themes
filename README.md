pycharm-themes
==============

TextMate themes for PyCharm
---------------------------

Installation for Mac OS X
=========================

```
$ git clone git://github.com/d1ffuz0r/pycharm-themes.git
$ cd pycharm-themes/colors
$ cp *.xml ~/Library/Preferences/PyCharm20/colors
```
